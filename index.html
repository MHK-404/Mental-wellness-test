Application Overview

MindWell is a user-friendly web application designed to help individuals assess and track their mental wellness. Users can take a short mental wellness test to receive immediate insights into their current well-being, categorized scores to better understand their mental state, and personalized recommendations for improvement. The application securely stores test results, allowing users to track their progress over time, and includes accessibility features to ensure inclusivity for all users.

Technology Overview

  • Frontend: HTML5, CSS3, JavaScript, React.js
  • Backend: Node.js, Express.js
  • Database: MongoDB
  • Authentication: Azure AD
  • Hosting: Microsoft Azure App Services
  • Additional Libraries: Chart.js (for progress visualization), Axios (for API calls)
