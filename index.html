Mental Wellness Test Proposal

Team Members

Mohammad Alkatheeri, Ali Alzaabi, Hussein Al-Rubeai, Mhd Saffouh Alajlani

Application Overview

The Mental Wellness Test helps users assess their mental state, providing scores and personalized recommendations.

Technology Overview

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js, Express.js
  • Database: Azure Storage
  • Hosting: GitHub Pages (Frontend), Azure App Services (Backend)

Feature List

  • Mental wellness test
  • Scoring system (Low, Moderate, High)
  • Personalized recommendations
  • Secure storage of test results
  • Accessible UI with screen reader support
